sogh

Root namespace for the SOG Helper Library

Methods

sogh.blurry (queries) static

Make Blurry Hidden Text

Name Type Description
queries String | Array CSS queries of elements which content has to be hidden
Example
sogh.blurry('div#container-id table tbody td:nth-child(6) a');

// Or
sogh.blurry([
    'div#container-id table tbody td:nth-child(6) a',
    'div#container-id table tbody td:nth-child(7)'
]);
